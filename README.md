# 簡易掲示板アプリ
インターンの課題で作成した簡易掲示板アプリです.

## 使用言語
* PHP/CSS/HTML
* MySQL

## アプリ概要
### ユーザ登録・ログインフォーム
掲示板ページの前にログイン或いはユーザ登録を行う必要がある
* ユーザ登録フォームは、メール認証によるユーザ登録機能を持つ
* ログインフォームでは、ユーザ登録フォームから登録したユーザでログインし、掲示板ページに遷移する機能を持つ

![ログインフォーム](https://github.com/ryuki999/WebTech/blob/master/img/ログインフォーム.PNG)


### 掲示板ページ
掲示板ページは、主に投稿・削除・編集フォームと投稿一覧部で分かれている.各役割は以下の通り
* 投稿フォームではコメントとパスワードを入力し、任意で画像や動画ファイルなどの投稿を行うと、IDはオートインクリメント、名前はログインユーザ名で新規投稿が行われる
* 削除フォームでは投稿IDとパスワードを指定すると、該当の投稿を削除することが出来る
* 編集フォームでは投稿IDとパスワードを指定すると、該当の投稿を編集することが出来る
* 投稿一覧部では投稿されたID・ユーザ名・コメント・画像or動画などを表示させている
![掲示板](https://github.com/ryuki999/WebTech/blob/master/img/掲示板.PNG)

### その他
* cssレスポンシブに設計されており、端末の画面サイズごとにレイアウトが変更される
* キャッシュ機能の実装がされており、2回目以降のページのロードではキャッシュを用いて表示が行われます
